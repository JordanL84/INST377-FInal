# HealthInfo Finder

## Description
HealthInfo Finder is a simple web application that helps users search for reliable health information. Users can enter a symptom, condition, or health topic, and the app returns educational results from the MedlinePlus Health Topics API. The app also saves recent searches to a Supabase database and displays a chart of recent search result counts.

## Target Browsers
This project is designed for current desktop browsers including:
- Google Chrome
- Safari
- Firefox
- Microsoft Edge

The application should also work on modern mobile browsers, but desktop browsers are the primary target.
